## How to Add an Entry / エントリの追加方法

When a mistake occurs, prepend a new entry (newest first) using this format:  
ミスが発生したら、以下の形式で先頭に追加する（新しい順）：

```markdown
### YYYY-MM-DD — <short title>
- **What went wrong:** ...
- **Root cause:** ...
- **Rule going forward:** <Yes/No checkable condition>
```
